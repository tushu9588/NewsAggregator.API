# News Aggregator API

Production-ready News Aggregator API built using ASP.NET Core.

## Features
- JWT Authentication
- Role-based Authorization (Admin/User)
- News CRUD APIs
- Pagination & Search
- MySQL + Entity Framework Core
- Swagger API Documentation

## Tech Stack
- ASP.NET Core Web API
- MySQL
- Entity Framework Core
- JWT
- Swagger
